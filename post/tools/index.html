Various Tools

Christoph Egerland
Last updated on Sep 19, 2018 1 min read

sent

  • tool for presentations
  • based on Takahashi method
  • minimal, easy to write

mathpix

  • screenshot of any equation to LaTeX
  • unfortunately only for Mac and mobiles There is a small workaround for Linux. Mathpix is based on an API that you can call up to 1000 times per month for free. So just get an API key, create a file eqtotex.py
#!/usr/bin/env python

# imports
import sys
import base64
import requests
import json

# file path of picture
file_path = str(sys.argv[1]);

# request api
image_uri = "data:image/jpg;base64," + base64.b64encode(open(file_path, "rb").read())
r = requests.post("https://api.mathpix.com/v3/latex",
    data=json.dumps({'src': image_uri}),
    headers={"app_id": "YOUR_APP_ID", "app_key": "YOUR_APP_KEY",
            "Content-type": "application/json"})
data = json.loads(r.text);

# print latex code
print data['latex']

and for laziness create a bash script called eq2tex in your PATH

#!/bin/bash

# execute python script with file as argument (e.g. eq.jpg)
echo `python2 /path/to/eqtotex.py $1`

Then if you have a picture of an equation, let’s say testeq.jpg, simply type into the terminal

$ eq2tex /path/to/testeq.jpg

and the LaTeX code of that equation is printed into the terminal.
