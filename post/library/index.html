Academic Library Management Done Right

Last updated on Sep 19, 2018 2 min read

It is really useful to keep a central place where you put all interesting things you find on the internet like Books, Websites, Articles, … . For me this central place is Zotero.

How my library is organized

I like keeping four sublibraries for

  • textbooks
  • my own calculations, transcripts and reviews
  • papers, articles, theses and similar
  • websites

in which I don’t have any further ordering.

Adding a new entry

The feature I like most about Zotero is the Zotero Connector. It’s an addon for firefox, that automatically senses content, so you can add it to Zotero with just one click. This is how I add new papers, books or websites usually.

In the other case I just add them manually to Zotero.

Citing in Latex

When I’m writing down some calculation or a draft I also need to reference work performed by others. I usually keep a long .bibtex file in some directory, which I’m linking in the .tex file of the calculation, draft or else. Any .bibtex file can be easily created with Zotero using Export > (Choose Better BibTex).

Citing other work could then look like this \cite{egerland17}. The short and more easy to handle label for a citation is called a bibtex key and there is a nice add-on in Zotero called Better BibTex which automatically generates them. My citation key format is [auth:lower][shortyear].

A central place for the files / Synchronization

One can synchronize the Zotero library if one creates an account. However a disadvantage of that is, that only the information, not necessarily the papers etc. get synchronized/backupped. Zotero also has the slight flaw of introducing its own storage system that is not just one folder with all the .pdfs and so on crammed into it.

A nice workaround for that is again an add-on, this one’s called Zotfile. It basically syncs the .pdfs to a folder that can be synchronized with other computers, e.g. a dropbox or owncloud folder.

I really like and use this add-on to have my papers and books with me on all my devices (PC at home, laptop, tablet, …).
