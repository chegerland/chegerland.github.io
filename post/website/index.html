How To: This Website

Last updated on Sep 19, 2018 2 min read

I have been searching long and wide for an easy to make and easy to maintain website. Most things sucked until I found out about R Markdown Websites. You basically write your content in markdown and then R Markdown handles everything for you. It was certainly a way to do it and for everyone who wants to try I can recommend:

  • Nick Strayer

  • Robert Mitchell

There were however some issues that bothered me. I couldn’t get the page to look the way I wanted it to look (which is probably my fault) and building the website took quite long, i.e. up to a minute. The webpage stood still for a couple of months until I discovered Hugo, which for now is the perfect solution for me.

Getting Started

The Quick Start shows how to install Hugo and deploy a first website with it. Just like my last webpage, this page is hosted on GitHub and there’s a nice manual on how to deploy Hugo as a GitHub Page. Furthermore I’m using the Academic Theme on this site (with some small modifications).

Managing Content

My workflow on this websites consists of starting the webpage with

$ hugo server

All the content is written in markdown files, which are easy to read and edit (for a good markdown tutorial see commonmark).

I add content, change content and reiterate until I’m happy. Saving the markdown files automatically rebuilds the site, so I can instantly see the changes.

To finish it off I’m using a script to deploy the page to GitHub.

$ ./deploy.sh "Commit message"

Icons in Hugo

I am using my favorite icon set Font Awesome. To incorporate them in Hugo one has to add

<script defer src="https://use.fontawesome.com/releases/v5.2.0/js/all.js" integrity="sha384-4oV5EgaV02iISL2ban6c/RmotsABqE4yZxZLcYMAdG7FAPsyHYAPpywE9PJo+Khy" crossorigin="anonymous"></script>

in /themes/YOUR_THEME/layouts/partials/header.html. There is also another way to embed icons, which I did not follow, mainly because I did not understand it.

To use font awesome also as the icon of the webpage (the small atom) I used fa2png.
